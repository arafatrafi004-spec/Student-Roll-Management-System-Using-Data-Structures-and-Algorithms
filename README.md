# 📚 Student Roll Management System (C++)

A simple console-based **Student Roll Management System** developed in **C++** to demonstrate the implementation of basic **Data Structures and Algorithms (DSA)**.

---

## 📌 Features

- Add Student Roll
- Display Student Rolls
- Bubble Sort
- Linear Search
- Binary Search
- Stack (Push & Pop)
- Queue (Enqueue & Dequeue)
- Binary Search Tree (BST)

---

## 🛠 Technologies Used

- C++
- Code::Blocks
- GNU GCC Compiler

---

## 📖 Data Structures & Algorithms

### Algorithms

- Linear Search

## ▶️ How to Run

1. Open the project in Code::Blocks.
2. Build the project.
3. Run the program.
4. Select an option from the menu.
